# SWAPI-PLANETS-REACT

## DEMO: https://0000marcell.github.io/swapi-planets-react/

how to install
```
git clone https://github.com/0000marcell/swapi-planets-react.git
cd swapi-planets-react
npm install
```

local server:
```
npm run start
```

tests:
```
npm run test
```

deploy:
```
npm run deploy
```
